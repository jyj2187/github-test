# Simple Todo App 제출용
일정 관리를 위한 Todo 웹 애플리케이션.

## 기능
- 편리한 UI로 쉽게 생성하고 쉽게 삭제하자.
- 기한과 카테고리를 직접 설정해보자
- create-react-app으로 간편한 번들링과 배포가 가능합니다.
- Spring Boot로 쉽게 서버 배포를 할 수 있습니다.
